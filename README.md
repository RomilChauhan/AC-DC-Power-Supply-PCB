## Overview

This project involves the complete design and physical construction of an **AC to DC power supply** — from initial circuit design through PCB layout in **Fusion 360**, board fabrication, component selection, manual assembly, and final functional testing. The result is a fully working power supply unit built entirely from scratch.

## Design Process

```
1. Circuit Design
      → Define input/output voltage requirements
      → Select transformer, rectifier, filter, regulator
      → Design full schematic

2. PCB Layout in Fusion 360
      → Import schematic netlist
      → Place components on board
      → Route power and signal traces
      → Define board outline, drill holes, copper pours
      → Design rule check (DRC)

3. Fabrication
      → Export Gerber files
      → PCB fabricated

4. Assembly
      → Source all components
      → Manual soldering — transformer, rectifier bridge,
        filter capacitors, voltage regulator, connectors
      → Inspect solder joints

5. Testing
      → Apply AC input
      → Measure DC output voltage
      → Verify regulation under load
      → Confirm full functionality
```

---

## Key Stages of an AC to DC Power Supply

```
AC Input → Transformer → Rectifier → Filter → Regulator → DC Output

[Step Down]   [AC→DC]    [Smooth]   [Regulate]
Transformer   Bridge     Capacitor  IC/Zener
              Rectifier  Filter     Regulator
```
